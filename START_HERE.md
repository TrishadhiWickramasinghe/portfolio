## 🎨 Customize in 3 Steps

### Step 1: Update Name (2 min)
Edit `app/page.tsx` and `components/Footer.tsx`
```
Find: "Your Name"
Replace: Your actual name
```

### Step 2: Add Projects (5 min)
Edit `app/projects/page.tsx`
- Add your 4+ projects to `allProjects` array

### Step 3: Update Links (3 min)
Edit various components:
- GitHub URL → Your GitHub
- LinkedIn URL → Your LinkedIn
- Email → Your email

**Done!** All other content auto-updates.
